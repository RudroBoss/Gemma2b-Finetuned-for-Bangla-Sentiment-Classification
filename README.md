# Gemma2b-Finetuned-for-Bangla-Sentiment-Classification
Fine-tuned Google Gemma-2B causal LM on a Bangla sentiment analysis dataset covering student, people, and mass uprising comments. Implements label masking, stratified train/eval split, optimized BF16 training, and disk-friendly checkpoints. Achieves 80% accuracy on test samples with quick inference.
